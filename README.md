# StudyOS — Smart Study Planner PWA

A fully offline-capable Progressive Web App for managing study sessions with timers, streaks, badges, and focus mode.

## Deploy to GitHub Pages

1. Push this folder contents to a GitHub repo's `main` branch
2. Go to **Settings → Pages → Source → Deploy from branch → main / (root)**
3. Visit `https://your-username.github.io/your-repo-name/`

The app works 100% offline after the first visit — no server needed.

## Features
- ▶ Start / ⏸ Pause / ✅ Complete manual task sessions
- ⏰ Auto-timer expiry with completion prompt
- 🔥 Daily streak tracking
- 🏅 Badges for achievements
- ◎ Distraction-free focus mode
- 🎮 Reaction game reward every 3 completed tasks
- 📲 Installable as a home screen app (PWA)
- 🌙 / ☀️ Dark / light theme
- 💾 All data stored in localStorage — no backend needed
